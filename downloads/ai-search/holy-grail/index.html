ONETOO AI Search — Holy Grail Reference Build

This is a static, signed, reference-only index build for Trust-First AI Search. It is published as a free-forever artifact for verification, education, and tooling integration.

Design principles
  • No runtime backend
  • No tracking, no auth, no accounts
  • Cryptographic integrity (SHA-256 + minisign)
  • Decade-stable URLs

Downloads

  • index-v1.zip
  • index-v1.zip.minisig
  • index-v1.zip.sha256.txt

Verification

Public key:

https://www.onetoo.eu/.well-known/minisign.pub

Verify (example):

minisign -V -p minisign.pub -m index-v1.zip -x index-v1.zip.minisig
